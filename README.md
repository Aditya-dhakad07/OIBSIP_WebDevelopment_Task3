# Temperature Converter

# Objective
To develop a Temperature Converter that converts temperature values between Celsius, Fahrenheit, and Kelvin using HTML, CSS, and JavaScript.

# Steps Performed:-
- Designed a clean and responsive user interface.
- Created input field for temperature values.
- Added options to select temperature units.
- Implemented temperature conversion logic using JavaScript.
- Displayed accurate conversion results instantly.

# Tools Used
- HTML5
- CSS3
- JavaScript
- Visual Studio Code

# Features
- Convert Celsius, Fahrenheit, and Kelvin
- Instant conversion
- Responsive Design
- Simple interface
- Input validation

# Outcome
Successfully developed a responsive Temperature Converter web application that accurately converts temperature values between Celsius, Fahrenheit, and Kelvin.
